# AnzenWorker

情報セキュリティにおいて重要となるエンドポイントセキュリティをより強固なものとするため、情報に関する知識が少ない人を対象として、実際に発生しうる攻撃やインシデントを仮想的に体験することで効率的に学習を進めるための体験型の学習システム。

### Installation
```bash
$ git clone https://github.com/NITMC-Procon/AnzenWorker.git
$ npm i -D
```

### Usage
```bash
$ npm start
```
